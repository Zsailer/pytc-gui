## quick terminal install

the pytc GUI is available for pip install:

	```
	pip install pytc-gui
	```

you can also clone from github and install from setup.py:

	```
    git clone https://github.com/harmslab/pytc-gui.git
    cd pytc-gui
    pip install .
    ```
